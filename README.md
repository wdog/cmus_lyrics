# CMUS_LYRICS

get lyrics from Genius and display in terminal


# REQUIREMENTS

* generate GENIUS API token and fill settings.json copying from
* secret.example.json


# USAGE


```bash
git clone git@github.com:wdog/cmus_lyrics.git
cd cmus_lyrics
python -m venv venv
pip install -r requirements.txt
./cmus_lyrics.py
```
